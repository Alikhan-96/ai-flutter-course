# Урок 3: Условные операторы и ветвление

## Основные темы

### Оператор if-else
```dart
int age = 18;

if (age >= 18) {
  print('Взрослый');
} else {
  print('Несовершеннолетний');
}
```

### Цепочка else if
```dart
double bmi = 23.5;

if (bmi < 18.5) {
  print('Недостаточный вес');
} else if (bmi < 25) {
  print('Нормальный вес');
} else if (bmi < 30) {
  print('Избыточный вес');
} else {
  print('Ожирение');
}
```

### Тернарный оператор
```dart
int age = 20;
String status = age >= 18 ? 'Взрослый' : 'Несовершеннолетний';
```

### Логические операторы
```dart
// && - логическое И
if (login == 'admin' && password == 'dart123') {
  print('Вход выполнен');
}

// || - логическое ИЛИ
if (age < 18 || !hasPermission) {
  print('Доступ запрещён');
}

// ! - отрицание
bool isActive = true;
print(!isActive); // false
```

### Вложенные условия
```dart
if (age >= 18) {
  if (hasLicense) {
    print('Можно водить');
  } else {
    print('Нужны права');
  }
} else {
  print('Слишком молод');
}
```

### Операторы сравнения
```dart
==  // равно
!=  // не равно
>   // больше
<   // меньше
>=  // больше или равно
<=  // меньше или равно
```

## Ключевые концепции
- Блоки кода определяются фигурными скобками `{}`
- Условие должно возвращать `bool`
- Проверка диапазона: `if (x >= 100 && x <= 999)`

## Типичные ошибки
- Использование `=` вместо `==` для сравнения
- Забытые `else` ветки
- Неправильный порядок условий в цепочке `else if`
