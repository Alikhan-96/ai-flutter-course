# Урок 2: Основы Dart - Переменные и типы данных

## Основные темы

### Функция main()
```dart
void main() {
  print('Hello, Dart!');
}
```

### Переменные и типы данных
```dart
// Явное указание типа
String name = 'Алихан';
int age = 25;
double height = 1.75;
bool isStudent = true;

// Вывод с интерполяцией
print('Имя: $name, Возраст: $age');
```

### Арифметические операции
```dart
int a = 10;
int b = 3;

print(a + b);  // Сложение: 13
print(a - b);  // Вычитание: 7
print(a * b);  // Умножение: 30
print(a / b);  // Деление: 3.333...
print(a ~/ b); // Целочисленное деление: 3
print(a % b);  // Остаток от деления: 1
```

### Комментарии
```dart
// Однострочный комментарий

/*
 * Многострочный
 * комментарий
 */

/// Документирующий комментарий
```

### Специальные символы в строках
```dart
print('Строка 1\nСтрока 2');  // Перенос строки
print('Табуляция:\tтекст');    // Табуляция
```

## Ключевые концепции
- **var** - автоматическое определение типа
- **final** - значение нельзя изменить после инициализации
- **const** - константа времени компиляции
- **Интерполяция строк** - `$variable` или `${expression}`

## Типичные ошибки
- Использование переменной до объявления
- Несоответствие типов при присваивании
- Отсутствие точки с запятой
